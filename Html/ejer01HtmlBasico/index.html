Seccion de etiquetas semanticas estructurales mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADRESS

Las etiquetas semanticas estructurales esran relacionadas directamente con el significado del contenido y no con los stilos ni las posiciones dentro del documento.

Reglas para etiquetas semanticas

  • Los distintos tipos de contenidp se pueden asociar a las etiquetas SECTION.
  • Las unidades de contenido se asocian a la etiqueta ARTICLE. O sea que un SECTION anida varios ARTICLES.
  • Las etiquetas HEADER y FOOTER pueden swer anidadas dentro de otras HEADER o FOOTER. Osea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elemnetos de navegacion del documento.
  • Reservar la etiqueta h1 para usar dentro de etiquetas top level como main, header o footer.
